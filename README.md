# REMATCH — Steam Analytics Case

Полный анализ 49 735 отзывов игры REMATCH (AppID 2138720, Sloclap): от сырых данных до A/B тестов, NLP, графиков и продукт-гипотез.

## Pipeline

```
steam_rematch_export.py   → сырые JSON из Steam API
process_analysis.py       → ETL в CSV + базовый отчет
deep_analysis.py          → A/B тесты, корреляции, 10 графиков
decline_analysis.py       → анализ спада онлайна, 12 гипотез
extra_analysis.py         → NLP (фичи, n-граммы), сегментация, выживаемость
generate_mega_dashboard.py → единый HTML-дашборд
```

## Что исследовано

| Область | Метрики |
|---|---|
| **Общие метрики** | 49 735 отзывов, 68.5% positive, 31.5% negative |
| **A/B тесты** | 8 тестов (плейтайм vs тональность, тип покупки, возвраты, Steam Deck, язык, время) + 4 корреляции Пирсона |
| **Онлайн** | Пик 91 374 (06/2025) → 6 201 (06/2026) = −93% за 12 мес (SteamCharts) |
| **Онбординг** | 76.9% негатива при плейтайме <30 мин (n=299); 62.1% при отзыве <30 мин (n=517) |
| **Выживаемость** | 50% игроков пишут отзыв (и часто уходят) до 10–20 часов плейтайма |
| **NLP** | Извлечение фич (читеры 72.7%, сервера 66.3%, монетизация 70.5%), n-граммы (game breaking, fix the game, dead game) |
| **Сегментация** | 21% энтузиасты, 9.6% долгие недовольные, 5.2% ушедшие новички |
| **Железо** | Топ GPU: RTX 5070 Ti (5.5% среди 401 отзыва с HW-данными), Windows 11 — 78.2% |
| **Достижения** | 88.9% имеют базовое достижение, 4.5% — все 37 (Rematch_AllTrophies) |

## Ключевые выводы

1. **Онбординг убивает быстрых уходящих** — среди игроков с плейтаймом <30 мин 76.9% оставляют негатив (vs 31.5% в среднем)
2. **Читеры** — 72.7% упоминаний в негативе, нет античита
3. **Контентные паузы** — 20 новостей за 55 недель, сезонные всплески быстро затухают
4. **Снежный ком** — чем меньше игроков, тем дольше матчмейкинг, тем быстрее уходят
5. **Нет удержания** — 4.5% всех достижений, нет ежедневных стимулов
6. **Разочарование после вложенного времени** — негативные отзывы пишут позже (≈67ч vs ≈43ч на момент отзыва), эффект «играл долго — разочаровался»

## Исправленные ошибки в данных

| Проблема | Было | Стало |
|---|---|---|
| «76.9% в первые 30 мин» без контекста | Путали с % положительных отзывов на **датском** (76.9%) | Явно: плейтайм <30 мин (n=299) и отдельно отзыв <30 мин (62.1%) |
| Chi-square в A/B отчёте | 50653.2% (ошибка форматирования ×100) | χ²=506.5, корректный p-value для таблицы 8×2 |
| GPU % | 5.3% (делили на все HW-записи) | 5.5% (делим на отзывы с указанной GPU) |
| RAM-график | Значения дублировались | Исправлен двойной append |
| SteamCharts график | Парсер JS не находил данные | Fallback на HTML-таблицу месяцев |

## Моё мнение об игре (по метрикам)

**REMATCH — это игра с сильным ядром и слабой экосистемой вокруг него.**

68.5% положительных отзывов при медианном плейтайме ~54 часа говорит о том, что **те, кто остаётся, искренне в неё верят**. 21% «энтузиастов» с 270+ часами — это редкий уровень вовлечённости для инди/AA-проекта. Механика, судя по отзывам и достижениям (56% дошли до ранга), цепляет футболом нового формата.

Но **−93% онлайна за год** — это не просто «естественный спад после хайпа». Это симптом системной проблемы:

- **Порог входа слишком высок** для casual-аудитории. 76.9% негатива у тех, кто не набрал и 30 минут — игра не успевает показать, за что её хвалят энтузиасты.
- **Live-service без live-ритма.** 20 новостей за 55 недель — это ~1 пост в 3 недели. Для соревновательного онлайна это медленно; паузы между сезонами убивают momentum.
- **Читеры (72.7% негатива в упоминаниях)** в PvP-футболе — dealbreaker. Без античита репутация «мертвой игры» (dead game) закрепляется сама собой.
- **Парадокс удержания:** негативные отзывы приходят от игроков с *бóльшим* playtime-at-review (67ч vs 43ч). Люди не уходят сразу — они *разочаровываются после инвестиции*. Это хуже, чем быстрый churn: теряется лояльное ядро.

**Вердикт:** REMATCH могла стать нишевым хитом уровня Rocket League в футбольной нише, но сейчас застряла в ловушке «мало игроков → долгий матчмейкинг → ещё меньше игроков». При этом качество для тех, кто остался, растёт (71.6% позитива в свежих отзывах vs 68.4% в старых) — команда Sloclap работает, но **новая аудитория не приходит**, а старая выгорает.

Что бы я сделал в первую очередь: античит + боты/быстрый матч при низком CCU + агрессивный F2P-weekend (бесплатные копии уже дают 69.8% позитива — триал работает). Без критической массы онлайна даже идеальный патч не спасёт.

## Технологии

- **Python** — сбор данных через Steam API, обработка, статистика
- **NumPy** — все вычисления (t-тест, z-тест, Cohen's d, Pearson r — ручная имплементация)
- **Matplotlib** — 18 графиков
- **HTML/CSS/JS** — интерактивный дашборд (base64-встроенные изображения, модальное окно)

## Запуск

```bash
python steam_rematch_export.py     # сбор данных
python process_analysis.py         # обработка
python deep_analysis.py            # A/B тесты + графики
python decline_analysis.py         # анализ спада
python extra_analysis.py           # NLP + сегментация
python generate_mega_dashboard.py  # финальный дашборд
```

Открыть `rematch_final_dashboard.html` в браузере.
