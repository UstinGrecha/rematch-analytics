# REMATCH — Steam Analytics Case

Полный анализ 49 735 отзывов игры REMATCH (AppID 2138720, Sloclap): от сырых данных до A/B тестов, NLP, графиков и продукт-гипотез.

## Pipeline

```
steam_rematch_export.py   → сырые JSON из Steam API
process_analysis.py       → ETL в CSV + базовый отчет
deep_analysis.py          → A/B тесты, корреляции, 10 графиков
decline_analysis.py       → анализ спада онлайна, 12 гипотез
extra_analysis.py         → NLP (фичи, n-граммы), сегментация, выживаемость
generate_mega_dashboard.py → единый HTML-дашборд
```

## Что исследовано

| Область | Метрики |
|---|---|
| **Общие метрики** | 49 735 отзывов, 68.5% positive, 31.5% negative |
| **A/B тесты** | 8 тестов (плейтайм vs тональность, тип покупки, возвраты, Steam Deck, язык, время) + 4 корреляции Пирсона |
| **Онлайн** | Пик 91 374 (06/2025) → 6 201 (06/2026) = −93% за 12 мес (SteamCharts) |
| **Выживаемость** | 50% игроков уходят до 10–20 часов плейтайма |
| **NLP** | Извлечение фич (читеры 72.7%, сервера 66.3%, монетизация 70.5%), n-граммы (game breaking, fix the game, dead game) |
| **Сегментация** | 21% энтузиасты, 9.6% долгие недовольные, 5.2% ушедшие новички |
| **Железо** | Топ GPU: RTX 5070 Ti (5.3%), Windows 11 — 78.2% |
| **Достижения** | 88.9% имеют базовое достижение, 4.5% — все 37 |

## Ключевые выводы

1. **Онбординг убивает** — 76.9% негатива в первые 30 минут
2. **Читеры** — 72.7% упоминаний в негативе, нет античита
3. **Контентные паузы** — 20 новостей за 55 недель, сезонные всплески быстро затухают
4. **Снежный ком** — чем меньше игроков, тем дольше матчмейкинг, тем быстрее уходят
5. **Нет удержания** — 4.5% всех достижений, нет ежедневных стимулов

| Файл | Описание |
|---|---|
| `rematch_final_dashboard.html` | Единый дашборд со всеми графиками, таблицами и выводами (1.4 МБ) |
| `rematch_export/` | Сырые JSON-данные из Steam API |

| `rematch_deep_analysis/` | A/B тесты, 10 графиков, глубокий отчет |

## Технологии

- **Python** — сбор данных через Steam API, обработка, статистика
- **NumPy** — все вычисления (t-тест, z-тест, Cohen's d, Pearson r — ручная имплементация, scipy was unavailable)
- **Matplotlib** — 18 графиков
- **HTML/CSS/JS** — интерактивный дашборд (base64-встроенные изображения, модальное окно)

## Запуск

```bash
python steam_rematch_export.py     # сбор данных
python process_analysis.py         # обработка
python deep_analysis.py            # A/B тесты + графики
python decline_analysis.py         # анализ спада
python extra_analysis.py           # NLP + сегментация
python generate_mega_dashboard.py  # финальный дашборд
```

Открыть `rematch_final_dashboard.html` в браузере.
